# Line Bot Side Project 想法管理幫手

幫你管理各種 side project 點子的小幫手  

## 功能
✅ 管理、記錄不同專案想法  
✅ 隨時隨地和 AI 討論專案，輕鬆在不同想法間切換  
✅ 根據過去的專案向你推薦新的靈感  
✅ 每天發通知提醒你討論專案  
🚧 查看和匯出各專案的對話紀錄
