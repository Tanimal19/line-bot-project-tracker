# Line-bot-project-tracker

line bot 專案想法管理幫手  
A simple project tracker using line bot.  

## 功能
- 組織、管理不同專案想法
- 隨時隨地和 AI 討論專案想法
- 每天自動發通知提醒
- 🚧 查看和匯出各專案的對話紀錄

開發日誌 [here](https://tanimalx.notion.site/Line-Pre-assessment-15957b83ce5b8076ba8de5f45839909c?pvs=4)
